Posted on 2018-08-16

闲来无事想写一个博客,因为自己有了 域名 ,但是想着写博客的话怎么也得用博客这个blog二级域名吧,博客也应该用hexo来写比较好,那怎么分配这个二级域名呢,查了半天发现nginx可以很好的实现,那还用什么apache啊,赶紧的nginx装起来,但是这个真的是个大坑……

首先安装nignx
sudo apt-get install nginx

接着安装php

sudo apt-get install php7.0

输入 php -v 出现版本信息就代表安装成功了

为什么是7.0的版本的 因为在nginx的配置文件里php的版本就是7.0的版本,为了方便

但是在站点下新建php文件,然后访问 你会发现它会把文件下载下来,这是因为nginx不能编译php文件,必须安装对应的插件

php低版本貌似要自己安装 7.0的版本好像自己有了 可能镜像不同会有差异 最好还是安装下一

sudo apt-get install php7.0-fpm

不确定这个是否是真的需要,我也没验证

sudo apt-get install php7.0-cgi

然后启动php服务

service php7.0-fpm start
